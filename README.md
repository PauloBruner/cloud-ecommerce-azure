# 🚀 Cloud E-Commerce Platform - Azure Powered

Projeto Full Cloud desenvolvido com Python e Microsoft Azure.

Aplicação web para cadastro e gerenciamento de produtos com imagens armazenadas na nuvem.

---

## 📌 Sobre o Projeto

Sistema Web construído com foco em integração Cloud, utilizando:

- ☁️ Azure SQL Database (armazenamento estruturado)
- ☁️ Azure Blob Storage (armazenamento de imagens)
- ⚡ Streamlit (interface web interativa)
- 🐍 Python (backend)
- 🐳 Docker (containerização)

---

## 🏗 Arquitetura

Usuário  
↓  
Streamlit App (Frontend + Backend Python)  
↓  
Azure SQL Database (Dados estruturados)  

Upload de imagem  
↓  
Azure Blob Storage  

---

## ⚙️ Funcionalidades

✔ Cadastro de produtos  
✔ Upload de imagens diretamente para o Azure Blob  
✔ Listagem dinâmica de produtos em formato de cards  
✔ Exclusão de produtos  
✔ Conexão segura com banco de dados  
✔ SQL parametrizado (proteção contra SQL Injection)  
✔ Variáveis protegidas via `.env`

---
